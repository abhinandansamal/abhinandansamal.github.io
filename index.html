Projects

Abalone Age Prediction using Machine Learning

Description: Developed a machine learning pipeline to predict the age of abalone from various physical measurements using advanced data preprocessing, feature engineering, and model training techniques. The project was executed as a comprehensive end-to-end solution from data preparation to model deployment, leveraging Python and several key data science libraries.

Key Features

Data Preprocessing: Cleaned and prepared the dataset consisting of 90,615 training samples and 60,411 test samples. This included handling missing values, outlier detection and removal, skewness correction, and data normalization.

Exploratory Data Analysis (EDA): Conducted thorough EDA to understand data distributions, relationships between variables, and to visualize key insights using matplotlib and seaborn libraries.

Feature Engineering: Created new features through transformations and interactions to enhance model performance. Applied one-hot encoding for categorical variables and standard scaling for numerical features.

Model Development: Implemented a Gradient Boosting Regressor as the primary model. Conducted hyperparameter tuning using GridSearchCV to identify the best parameters, improving model accuracy.

Model Evaluation: Evaluated model performance using Root Mean Squared Logarithmic Error (RMSLE) and R-squared metrics. Achieved a validation RMSLE of 0.1425 and a validation R2 of 0.6783, indicating robust model performance.

Logging and Exception Handling: Integrated comprehensive logging and exception handling to ensure smooth execution and ease of debugging.

Visualization: Generated detailed visualizations, including histograms, pair plots, and correlation matrices to support data-driven decisions and model interpretations.

Deployment: Prepared the final model for deployment using joblib for model serialization and Docker for containerization, ensuring reproducibility and scalability.

Technologies Used

Programming Languages: Python (NumPy, pandas, matplotlib, seaborn, scikit-learn, joblib)

Data Processing: Data cleaning, outlier detection, Winsorization, data normalization

Machine Learning: Gradient Boosting Regressor, hyperparameter tuning (GridSearchCV)

Model Evaluation: RMSLE, R-squared

Tools and Frameworks: Jupyter Notebook, Docker

Software Development Practices: Modular code structure, configuration management using YAML, logging, and exception handling

Walmart Sales Forecasting using Machine Learning

Description: Developed a machine learning model to forecast the daily sales for the next 28 days at Walmart stores in three US states (California, Texas, and Wisconsin). The primary objective is to improve forecast accuracy using machine learning techniques, specifically targeting improvements in inventory management by reducing overstock and stockout scenarios.

Key Features

Data Preprocessing: Preprocessed and analyzed 6M+ rows of data. This included merging datasets, handling missing values, and ensuring data consistency.

Feature Engineering: Extracted day, month, year, and holiday flags. Created lag features to capture past sales trends. Applied one-hot encoding to categorical features such as event names and types. Ensured both training and evaluation datasets had the same features.

Model Development: Implemented Prophet, an open-source forecasting tool by Facebook, designed for handling daily seasonality, holidays, and trend changes, as the model.

Model Evaluation: Evaluated model performance using Mean Absolute Error (MAE) and Root Mean Squared Scaled Error (RMSSE) metrics. Achieved a validation MAE of 2.44 and a validation RMSSE of 0.6054, indicating robust model performance.

Business Impact: Calculated the average selling price of the items as $3.66. Estimated the financial savings of $250.44 from improved forecasting accuracy.

Logging and Exception Handling: Integrated comprehensive logging and exception handling to ensure smooth execution and ease of debugging.

Visualization: Generated detailed visualizations, including distribution of sales for a specific item and forecast plot.

Technologies Used

Programming Languages: Python (NumPy, pandas, matplotlib, seaborn, statsmodels, Prophet, joblib)

Data Processing: Data merging, date features extraction, lag features creation, one-hot encoding, alignment of training and evaluation datasets

Machine Learning: Prophet Model

Model Evaluation: Mean Absolute Error (MAE), Root Mean Squared Scaled Error (RMSSE)

Tools and Frameworks: Jupyter Notebook, Google Cloud, Google Colab

Software Development Practices: Modular code structure, configuration management using YAML, logging, and exception handling
